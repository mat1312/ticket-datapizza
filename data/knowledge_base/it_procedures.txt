PROCEDURE IT AZIENDALI - GUIDA OPERATIVA

=== ACCESSO VPN ===

CONFIGURAZIONE INIZIALE:
- Scaricare il client OpenVPN dal portale aziendale (portal.company.com)
- Utilizzare le credenziali Windows aziendali per l'autenticazione
- Server VPN: vpn.company.com:1194
- Protocollo: OpenVPN con certificato aziendale

RESET PASSWORD VPN:
1. Andare su portal.company.com → Gestione Account
2. Selezionare "Reset Password VPN"
3. Inserire il proprio numero di telefono aziendale
4. Riceverai un SMS con il codice temporaneo
5. Utilizzare il codice per impostare una nuova password
6. La password deve contenere: 8+ caratteri, maiuscole, minuscole, numeri, simboli

TROUBLESHOOTING VPN:
- Errore "Timeout connessione": Verificare firewall Windows e antivirus
- Errore "Certificato scaduto": Contattare IT per rinnovo certificato
- Errore "Autenticazione fallita": Verificare username/password e fare reset se necessario
- Performance lenta: Provare server alternativo vpn2.company.com:1194

=== PROBLEMI EMAIL ===

OUTLOOK NON SINCRONIZZA:
1. Chiudere completamente Outlook
2. Andare in Pannello di Controllo → Account di Posta
3. Testare impostazioni account
4. Se errore, rimuovere e riconfigurare l'account
5. Server: mail.company.com, Porta: 993 (IMAP) o 995 (POP3)

CASELLA PIENA:
- Quota massima: 2GB per utente
- Controllare cartella "Elementi eliminati" e svuotarla
- Archiviare email vecchie di oltre 6 mesi
- Utilizzare OneDrive per allegati grandi (>10MB)
